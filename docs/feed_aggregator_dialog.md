# Feed aggregator dialog

This dialog shows the progress of the feed aggregator (see [Feeds and
subscriptions](feeds_and_subscriptions.md)).  It allows you to start and stop
the aggregation process, although stopping might take some time,
because it has to cancel a background thread.

The main purpose of the Feed aggregator dialog is to troubleshoot the
aggregation process. It doesn't do anything differently from
_Subscriptions|Update subscriptions_, but it shows how long each feed takes to
download and process, and how many posts are being found from each feed.

You can open the Feed aggregator dialog whilst the feed aggregator is still
running, but it will only show new updates, not previous ones.
