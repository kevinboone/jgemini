# How to report a bug

It's best to report bugs through JGemini's
[GitHub repository](https://github.com//kevinboone/jgemini). If practicable, please 
provide enough information to reproduce the bug, including details of the 
Java JVM and platform. 

Please include the exact version number of JGemini, from Help|About or the version
number on its JAR file.

In many cases it will help to run JGemini with maximum logging, and redirect
program output to a file.
